  • Authorization: Advanced Role-based access control.
  • Clean Architecture: Implementation of Repository & Service patterns.
  • Validation: FluentValidation & AutoMapper integration for clean data flow.
  • Stack: C#, EF Core, ASP.NET Identity, SQL Server, Swagger, Git.
